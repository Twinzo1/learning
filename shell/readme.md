### 分割字符串
* 以空格分割字符串```echo "123" |sed 's/./& /g```
### ```[ $var -eq "1" ]```
* 当var为空时会报错````bad number```，改为```[ "$var"x = "1"x ]```
### ```&&```与```||```区别
* ```&&```：前面有一个为真时，就会执行；```||```：只看左边一个，左边为假则执行
